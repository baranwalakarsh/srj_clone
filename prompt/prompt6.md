# SRJ Global Technologies Website Clone - Final Implementation Specification

## Objective

Rebuild the website so that it visually and structurally matches the reference website:

https://www.srjglobalsoftech.com/

The goal is not to create a similar website. The goal is to replicate the structure, layout hierarchy, spacing, visual appearance, sections, animations, cards, footer, navigation, and page organization as closely as possible.

---

# Global Rules

## Important

* Use the reference website as the source of truth.
* Match spacing, typography, colors, card layouts, section ordering, and responsiveness.
* Do not invent new sections.
* Do not add extra content.
* Do not remove any content that exists in the reference site.
* Reuse the same visual hierarchy throughout the project.
* Match desktop and mobile layouts.
